# Bilder / Fotos

Legt hier eure eigenen Fotos ab (z. B. vom Unihockeyfeld beim Kastanienhof) und
verlinkt sie in `../config.js`:

```js
HERO_IMAGE: "images/feld.jpg",            // grosses Hintergrundbild im Hero
GALLERY:   ["images/feld-1.jpg",          // optionale Galerie weiter unten
            "images/feld-2.jpg"],
```

Empfohlen:
- **Hero-Bild:** quer (Landscape), mind. 1600 px breit, JPG.
- **Galerie:** beliebig, werden automatisch zugeschnitten.

Solange hier nichts liegt, zeigt der Hero einen Farbverlauf und die Galerie bleibt
ausgeblendet – die Seite funktioniert also auch ohne Fotos.
